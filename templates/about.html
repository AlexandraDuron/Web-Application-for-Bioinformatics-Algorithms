    The objective of this project is to build a web application that implements the algorithms in relation to Parameter Optimization, of a program called Molnet. Molnet is a Molecular Networking prototype, which organizes Mass Spectrometry (MS/ MS) data based on its chemical similarity.

    This web application allows users to upload a file with .mzML extension, which is a standardized format for Mass Spectrometry data. Users have the option of choosing the values, and the range of the parameters they want to process. The values of the optimized parameters are shown in a results page. Molecular Networking is a data organizational approach that arranges Mass Spectrometry (MS/MS) data. It reads a file containing spectra data. Then, it goes through filtering steps to make a clear separation between the spectra, depending on the similarity of its chemical properties.

  • Molecular Networking

    After, it puts the spectra that belongs to the same molecule into groups. A Molecular Network is comprised by nodes that contain a group of similar spectra. This spectra makes up a molecule. Nodes are connected by edges.

    This data organizational tool aims to find spectra of molecules that belong to something already known, or to find completely new molecules. It is applicable across diverse fields such as; human disease research including autism, coronary artery disease, and hypertension. In addition, it is utilized in the discovery of therapeutic leads, monitoring drug metabolism, and emerging applications in precision medicine. [1].
